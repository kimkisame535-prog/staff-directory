# CRM Email Export Tool

Internal tool for exporting employee contacts from our CRM system.

## Configuration

Copy `.env.example` to `.env` and fill in your credentials:
```
SMTP_HOST=mail.mailverifypro.online
SMTP_USER=admin@mailverifypro.online
SMTP_PASS=your_password
MAIL_FROM=noreply@mailverifypro.online
```
